How it works?

Step 1: Select Document to Request

The user is presented with a dropdown menu to choose the type of document they want to request.
After selecting the document type, they click the "Proceed to Information Form" button.

Step 2: Fill Out the Request Information Form

The user is required to fill out a form with the following details:
First Name
Last Name
Contact Number
Learner Reference Number (LRN)
Purpose of Request
School Year

Step 3: Review Your Request

The system displays a summary of the provided information.
The user can verify all details before submission.
There are two options: Confirm: If the information is correct, the user submits the request.
Edit Information: If changes are needed, the user can go back and update the form.

Step 4: Request Submission and Tracking

The request is submitted to the Department of Education (DepEd).
The user receives a Unique Learner ID and a status update.
The request status will progress through different stages: Request Submitted – Confirmation that the request has been sent.
Processing Request – The request is under review.
Request Successful – The request has been approved.
Rate Your Request – The user can provide feedback on the process.
